# 宠物领养管理系统 java938

## 项目概述

宠物领养管理系统是一个基于前后端分离架构的Web网站，主要使用Vue和Java技术栈开发。系统旨在提供一站式的宠物领养服务，包括流浪动物管理、领养申请处理、宠物救助等功能。

## 技术栈

- 前端：Vue
- 后端：Java、SpringBoot、Mybatis
- 数据库：Mysql
- 开发工具：IDEA、JDK1.8、Maven
